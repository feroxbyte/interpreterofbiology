# The Interpreter of Biology — Landing Page

Single-page marketing site for a science-first biology consultation service.
Built with Astro + Tailwind v4. Brand system from `interpreter of biology brand guide v11.html`.

## Develop
- `pnpm dev` — start dev server at http://localhost:4321
- `pnpm build` — production build to `dist/`
- `pnpm preview` — preview the build

## Theming
All brand tokens (colors, fonts, hard-offset shadows) live in `src/styles/theme.css` under `@theme`.
